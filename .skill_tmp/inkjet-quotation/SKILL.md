# สกิลเสนอราคางานพิมพ์อิงค์เจ็ท (Wide Format)

ข้อมูลวัสดุครบทุกประเภทอยู่ใน `references/inkjet-materials.md`
ราคาอ้างอิงและโครงสร้าง Tier Pricing อยู่ใน `references/inkjet-pricing.md`

---

## ขั้นตอนคำนวณราคา

### ขั้น 1 — รับข้อมูลงาน

| ข้อมูล | ตัวอย่าง |
|---|---|
| ประเภทวัสดุ | ไวนิล Outdoor / สติ๊กเกอร์ PVC / PP Paper |
| ขนาดชิ้นงาน | กว้าง × สูง (เมตร หรือ ซม.) |
| จำนวนชิ้น | 10 ป้าย |
| ความละเอียด | 720 dpi (ทั่วไป) / 1440 dpi (ละเอียดสูง) |
| งานด่วน? | ใช่ / ไม่ใช่ |
| บริการเสริม | ตัด, พับขอบ, เจาะตาไก่, ร้อยท่อ, ติดตั้ง |

---

### ขั้น 2 — คำนวณพื้นที่

```
พื้นที่ 1 ชิ้น (ตร.ม.) = กว้าง (ม.) × สูง (ม.)
พื้นที่รวม (ตร.ม.)   = พื้นที่ 1 ชิ้น × จำนวนชิ้น
```

> **ระวัง:** หากมี Bleed ให้บวก 0.05 ม. (5 ซม.) ต่อด้าน

---

### ขั้น 3 — เลือกราคาตาม Tier

ดูตาราง Tier Pricing ใน `references/inkjet-pricing.md`

**ตรรกะ Tier:**
- ไวนิลและสติ๊กเกอร์ PVC: มี Volume Discount ตามพื้นที่รวม
- วัสดุอื่น ๆ: ส่วนใหญ่คิดราคาเดียว (ไม่ขึ้นกับปริมาณ)
- งานด่วน: บวกเพิ่ม 20–100 บาท/ตร.ม. ตามวัสดุ

---

### ขั้น 4 — บวกค่าบริการเสริม (Finishing)

| บริการ | ราคาอ้างอิง |
|---|---|
| ตัดตามขนาด | รวมในราคา (งานน้อย) / แยกคิด (งานมาก) |
| พับขอบ + ร้อยท่อ | รวมในราคา 1–49 ตร.ม. |
| เจาะตาไก่ (ตีตาไก่) | รวมทุก 1 เมตร (งาน 1–49 ตร.ม.) |
| ท่อ PVC + เชือก | คิดแยก |
| ติดตั้ง | คิดแยกตามพื้นที่/สถานที่ |
| เคลือบฟิล์มป้องกัน | บวกเพิ่มตามประเภทฟิล์ม |

## ตัวอย่างคำนวณ

### ตัวอย่าง 1: ป้ายไวนิล 3×1 เมตร จำนวน 5 ป้าย (1440dpi)

```
พื้นที่ = 3 × 1 = 3 ตร.ม./ชิ้น
รวม    = 3 × 5 = 15 ตร.ม.  →  เรท 1–49 ตร.ม.
ราคา   = 15 × 150 = 2,250 บาท
(รวมตัด พับขอบ เจาะตาไก่ทุก 1 ม.)
ราคา/ชิ้น = 2,250 ÷ 5 = 450 บาท/ป้าย
```

### ตัวอย่าง 2: สติ๊กเกอร์ PVC ติดผนัง 1×1 เมตร จำนวน 200 ตร.ม. (720dpi)

```
พื้นที่รวม = 200 ตร.ม.  →  เรท 100–499 ตร.ม.
ราคา = 200 × 100 = 20,000 บาท (ไม่รวมค่าตัด/ติดตั้ง)
```

### ตัวอย่าง 3: ผ้า IT Backdrop 4×2 เมตร 1 ผืน (1440dpi)

```
พื้นที่ = 4 × 2 = 8 ตร.ม.  →  เรท 1–49 ตร.ม.
ราคา   = 8 × 300 = 2,400 บาท
```
