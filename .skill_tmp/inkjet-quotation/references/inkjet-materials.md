## หมวด 1: Vinyl / ไวนิล (PVC Flex)

### 1.1 Vinyl Frontlit — แบนเนอร์ทึบแสง
| รายการ | ค่า |
|---|---|
| วัสดุ | PVC Flex ทึบแสง |
| น้ำหนัก | 440 gsm, 510 gsm, 680 gsm |
| ความกว้างม้วน | 0.9 / 1.2 / 1.5 / 1.8 / 2.0 / 2.5 / 3.0 / 3.2 / 5.0 ม. |
| หมึกที่ใช้ได้ | Solvent, Eco-Solvent, Latex |
| อายุการใช้งาน Outdoor | 2–5 ปี |
| ใช้กับ | ป้ายหน้าร้าน, แบนเนอร์, ป้ายงาน, Billboard |

### 1.2 Vinyl Backlit — แบนเนอร์ป้ายไฟ
| รายการ | ค่า |
|---|---|
| วัสดุ | PVC โปร่งแสง พิเศษสำหรับแสงส่องผ่าน |
| น้ำหนัก | 440–510 gsm |
| ความกว้างม้วน | 1.0–5.0 ม. |
| ใช้กับ | Light Box, ป้ายไฟ, Totem, Backlit Signage |

### 1.3 Vinyl Blockout — กันแสงทะลุ

|---|---|
| วัสดุ | PVC ดำด้านหลัง (Black Back) |

## หมวด 7: จำแนกตามประเภทหมึก

| ประเภทหมึก | วัสดุที่รองรับ | ใช้กับเครื่อง |
|---|---|---|
| **Aqueous (น้ำ)** | Photo paper, Fine Art, Canvas, PP Paper | Epson, Canon, HP (Desktop/Photo) |
| **Eco-Solvent** | Vinyl, Sticker PVC, Canvas, Fabric | Roland, Mimaki, Mutoh |
| **Solvent** | Vinyl Outdoor ขนาดใหญ่, Banner Flex | Flora, Wit-Color |
| **Latex** | Vinyl, Sticker, Fabric, Wallpaper, Canvas | HP Latex, Epson SureColor |
| **UV-Curable** | Rigid Board, PVC, Acrylic, Metal, Wood, กระจก | Roland VersaUV, Mimaki UJF |

## หมวด 8: ขนาดความกว้างมาตรฐาน

| Format | ความกว้าง | วัสดุที่ใช้บ่อย |
|---|---|---|
| Small Format | 0.3–0.6 ม. | สติ๊กเกอร์ฉลาก, ป้ายเล็ก |
| Medium Format | 0.6–1.6 ม. | โปสเตอร์, สติ๊กเกอร์ผนัง |
| Wide Format | 1.0–3.2 ม. | ป้ายไวนิล, แบนเนอร์ |
| Super Wide | 3.0–5.1 ม. | Billboard, ป้ายใหญ่ |

## หมวด 9: Indoor vs Outdoor

| วัสดุ | Indoor | Outdoor |
|---|---|---|
| Vinyl Frontlit | ✅ | ✅ (2–5 ปี) |
| Vinyl Backlit | ✅ | ✅ |
| Sticker PVC ขาว | ✅ | ✅ |
| Sticker PVC ใส | ✅ | ✅ (แยกเกรด) |
| Sticker 3M | ✅ | ✅ (ทนทานสูง) |
| Sticker Seethrough | ❌ | ✅ |
| PP Paper | ✅ | ❌ |
| Glossy/Matte Paper | ✅ | ❌ |
| Fine Art Paper | ✅ | ❌ |
| ผ้า IT | ✅ | ✅ |
| ผ้า Mesh | ❌ | ✅ |
| Canvas | ✅ | ⚠️ (ต้องเคลือบ) |
| Wallpaper | ✅ | ❌ |
| Floor Graphic | ✅ | ✅ |
| Reflective Film | ❌ | ✅ |
